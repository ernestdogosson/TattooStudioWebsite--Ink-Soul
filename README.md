# Tattoo Studio Website -- Ink & Soul

A collaborative tattoo studio website built to practice multi-page navigation, custom SVG design, and business website development. This group project helped us understand how to create professional websites for real businesses.

## Project Overview

Ink & Soul Tattoo Studio is a complete business website where we focused on creating custom branding, organizing multiple pages, and making everything responsive. We wanted to build something that looked professional and could actually be used by a real tattoo studio.

This project taught us how to think about business needs, not just technical implementation.

## Main Features

- Custom SVG logo with gradient effects
- Multi-page navigation (Home, About, Gallery, Contact)
- Responsive design for all devices
- Professional typography with custom fonts
- Social media integration
- Business information and contact details
- Organized CSS structure
- Accessibility features

## Technologies Used

- HTML5
- CSS3 -- Flexbox, Grid, animations
- SVG Graphics -- custom logo design
- Font Awesome -- social media icons
- Google Fonts -- Kaushan Script, New Rocker, Roboto
- Normalize.css -- cross-browser consistency

## Design Highlights

- Custom SVG logo with gradient effects
- Dark theme suitable for tattoo industry
- Three complementary font families
- Professional imagery and icons
- Responsive layout for all devices

## My Contributions

- Design & Prototyping: Created complete website design in Figma, including layout mockups, color schemes, and typography choices
- Custom SVG Logo: Designed and built the Ink & Soul logo from scratch using SVG with custom gradients and effects

- Navigation System: Created the multi-page navigation structure

## Responsive Design

- Mobile -- optimized navigation and touch interface
- Tablet -- adjusted layouts and spacing
- Desktop -- full-featured experience with hover states
